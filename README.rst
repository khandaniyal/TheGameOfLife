javac 'nombre'.java.
y para ejecutarlo con un java 'nombre'.
```

practice.java  README.rst
diego.mendez@a25pc0:~/TheGameOfLife$ javac practice.java
diego.mendez@a25pc0:~/TheGameOfLife$ java practice.
```
